## 프로젝트 개요
KETCHUP은 기업 내 업무 공유와 협업을 원활하게 하여 업무의 효율성을 높이고, 결과적으로 기업의 생산성 향상에 크게 기여합니다.

**Restful API** 설계원칙을 기반으로 개발된 그룹웨어 프로젝트입니다.

## 주요기능
- **사내 커뮤니케이션**: 이메일, 게시판, 공지 사항 기능으로 조직 내 원활한 의사소통을 지원합니다.
- **자료 중앙 관리**: 부서별 게시판을 통해 자료와 문서를 일원화하여 저장하고 관리함으로써 업무 효율성을 향상시킵니다.
- **업무 프로세스 표준화**: 조직 구성원들이 동일한 전자결재 양식을 사용하여 의사결정을 체계적으로 처리할 수 있습니다.


## 담당사항
- **SpringSecurity**와 **JsonWebToken**을 이용한 **로그인,회원가입** 구현
- Member,Department,Position API 구현
- **SpringBoot Starter Mail**을 사용한 이메일 발송 구현
- **React-D3-Tree**를 사용한 조직도 구현
- **Procedure**를 활용한 DB 내부 자동 삭제 구현
- **다음 주소 API**를 활용한 우편주소 검색 구현
